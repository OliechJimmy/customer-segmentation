##Customer Segmentation with RFM Analysis

An e-commerce data science project that segments 4,339 customers 
into behavioural groups using Recency, Frequency and Monetary analysis.

 What it does
- Cleans and prepares 500,000+ retail transactions
- Calculates RFM metrics for every unique customer
- Scores and segments customers into 5 groups:
  Champions, Loyal Customers, Potential Loyalists, At Risk, Lost
- Visualises segment distribution and average spend per segment
- Generates business recommendations per segment

## Key Findings
- 1,268 Champions identified as highest value customers
- 988 At Risk customers flagged for win-back campaigns
- Champions have the highest average monetary value confirming scoring accuracy
